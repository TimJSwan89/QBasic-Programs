Loops
Loop are a powerful technique that give the programmer the ability to have the program repeat code.  Repeating code saves time for the programmer, save space in the program, and makes programs easier to read (more compact code).  There are three types of loops that we will learn about.  Each loop has a particular function with certain advantages.
1.  For Next loop

The For-Next loop is designed to run a set number of times.  Useful when you want a loop to run exactly a certain number of times.

Enter and run the following

    PRINT "Enter three numbers"

    For x = 1 to 3
        INPUT Num
    Next x

 Note that the loop allows you to enter three numbers but doesn't do anything with the numbers.

Remember to indent your code within a loop (easier to read).  As well, put a line of white space before and after the loop.
2.  For Next loop 

Let's change the loop in number 1 to add the numbers and prompt the user for each number (red code).

    For x = 1 to 3
        Print "Enter number " ; x ; ": ";
        INPUT Num
        Sum = Sum + Num
    Next x

 Now the program will ask for each number.  The line Sum = Sum + Num is how we add the numbers.  As the loop runs each time, Sum will add to itself the new Num entered. We are still missing part of the program?

4.  Using For Next to draw graphics

We can use the For Next to draw using QBasic graphics mode (Screen 12).

Enter the following to draw rectangles

    screen 12

    For x = 0 to 30 STEP 10
        Line (220 + x, 140 + x) - (420 - x, 140 + x)
        Line (420, 140) - (420, 340)
        Line (420, 340) - (220, 340)
        Line (220, 340) - (220, 140)
    Next x

 *  The code will only draw one rectangle and then the top three lines for the inner rectangles.

*  The loop is counting up by 10's starting at 0 and going to 30.  That will give us a total of four rectangles.  In effect, the loop adds/subtracts depending how you set up each column/row number.

*  You have to use the x to add/subtract from the column/row numbers to get the loop to draw the lines properly.

 
5.  Do-Loop Until

This loop will run at least once and will stop when the condition is met.  Enter the following at run.

     Print "Guess a number 1-10"

     Do
          Print "Enter guess" ;
          Input Guess
          If Guess <> 2 Then
              Print "Incorrect"
          End If
     Loop Until Guess = 2

     Print "Correct!"

 The condition to quit this loop is Guess = 2.  You can set up any condition you want.  Remember, the loop runs UNTIL the condition is True, then the loop stops and the program continues.

6.  While - Wend loop

The While loop may not run at all depending on the initial condition.  Enter and run.

    Print "Enter your favourite colour";
    Input Colour$

    While Colour$ <> "red"
         Print "Wrong Colour, try again";
         Input Colour$
    Wend

     Print "That is the correct colour!"

 Note that the condition is at the beginning of the loop.  If the user enters red at the start, the loop will never run.  The loop will only run if the condition is True.
